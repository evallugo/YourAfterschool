Import Inventory Items

Option 1: Upload Spreadsheet

Upload a CSV or Excel file with your existing inventory. The system will match columns automatically.

Option 2: Auto-create from Lesson Lists

Automatically create inventory records for every unique item that appears in your imported lesson packing lists. Quantities will be set to 0 — you then update the actual counts.

Best for: Getting your item list set up quickly after importing the master sheet. You'll still need to count and enter quantities.
