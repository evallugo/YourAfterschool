What happens after import
  1. Each unique Class + Lesson Name + Lesson # combination becomes a lesson in the database.
  2. All items for that lesson are stored and trackable in the packing checklist.
  3. Items in the Return column are highlighted on packing sheets.
  4. An Excel file with one tab per lesson is generated for download.
  5. Re-importing the same sheet updates existing lessons without creating duplicates.
